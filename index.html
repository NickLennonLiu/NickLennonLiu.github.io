FSE 2026 · Industry Lead Author

Eagle: Leveraging Operations Documents for Comprehensive Benchmark Question Generation

Yuhe Liu, Changhua Pei, Hang Wang, Longlong Xu, Xiaogang Dong, Zhen Feng, Li Zheng, Kehang Ji, Dan Pei

An end-to-end framework deployed inside Huawei that ingests enterprise product documentation and synthesizes 4,845 domain-grounded QA pairs across logs, metrics, traces and configurations. Eagle improves expert-rated rubric scores by 22%–49% over SOTA baselines.

Conference page Code Dataset
FSE 2025 Lead Author

OpsEval: A Comprehensive Benchmark Suite for Evaluating Large Language Models' Capability in IT Operations Domain

Yuhe Liu, Changhua Pei, Longlong Xu, Bohan Chen, Mingze Sun, Zhirui Zhang, Yongqian Sun, Shenglin Zhang, Kun Wang, Haiming Zhang, Jianhui Li, Gaogang Xie, Xidao Wen, Xiaohui Nie, Minghua Ma, Dan Pei

A comprehensive benchmark with 9,000+ AIOps QA pairs across 8 sub-domains, multi-paradigm evaluation (zero/few-shot, CoT, self-consistency) and an expert-aligned QA metric (FAE-Score) with 0.9185 correlation to human review.

arXiv:2310.07637 Leaderboard Dataset
IJCNN 2025 3rd Author

TechSupportEval: An Automated Evaluation Framework for Technical Support Question Answering

Bohan Chen, Yongqian Sun, Yuhe Liu, Longlong Xu, Zhe Xie, Changhua Pei, Jing Han, Fan Ni, Xuhui Cai, Ce Yang, Dan Pei

An automated evaluation framework for technical-support QA that combines ClozeFact (cloze-style fact verification) and StepRestore (step ordering reconstruction). Achieves AUC 0.91 on a TechQA-based benchmark — +7.6% over prior SOTA.

Paper Code DOI
IEEE TIP 2023 2nd Author

Skeleton-CutMix: Mixing Up Skeleton with Probabilistic Bone Exchange for Supervised Domain Adaptation

Hanchao Liu, Yuhe Liu, Tai-Jiang Mu, Shi-Min Hu

A cross-domain skeleton augmentation method for action recognition transfer. By probabilistically swapping bones between source and target skeletons, the model generalizes across NTU-60 and ETRI-Activity3D datasets.

Code IEEE TIP · Vol. 32, 4046–4058
