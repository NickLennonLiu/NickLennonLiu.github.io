# Yuhe Liu Homepage

Static personal homepage for `NickLennonLiu.github.io`.

The blog is deployed separately from `NickLennonLiu/river-blog-deploy` and uses the custom domain `https://blog.river9.top/`. The local `/blog/` path redirects there because GitHub Pages project paths follow repository names; the `river-blog-deploy` repository would otherwise publish under `/river-blog-deploy/`.
